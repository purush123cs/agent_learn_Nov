# LangChain GetWeatherHardcoded

## Setup

This uses pip and not uv.
Usual pip commands can be used

### Create a virtual environment using the command:

`python3 -m venv <environment_name>`

### Activate the Virtual env (On Mac):

`source <environment_name>/bin/activate`

### Installing dependencies from requirements.txt.

`pip install -r requirements.txt`

This command reads the requirements.txt file and installs all listed packages and their specified versions.

### Run the program:

`python getWeatherBasic_Gemini.py`



### Deactivate the Virtual env:

`deactivate`

### Installing Packages within a Virtual Environment:

`pip install <package_name>`

### Listing installed packages:

`pip list`

### Deleting a Virtual Environment:

`rm -rf <environment_name>`  # macOS/Linux

### Generating requirements.txt.

`pip freeze > requirements.txt`

This command captures all currently installed packages in your active Python environment and writes them to requirements.txt.
